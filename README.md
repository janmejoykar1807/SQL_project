# SQL Project

A database design project built using MySQL Workbench. Contains an Entity-Relationship (ER) model representing a structured relational database schema.

---

## 📋 Overview

This project demonstrates relational database design principles, including entity modeling, relationships, primary/foreign keys, and schema normalization. The model was built using **MySQL Workbench** and exported as a `.mwb` file.

---

## 📁 Repository Structure

| File | Description |
|------|-------------|
| `Final_model.mwb` | MySQL Workbench database model — full ER diagram and schema |

---

## 🛠️ Technologies Used

- **MySQL Workbench** — ER diagram design and schema modeling
- **SQL** — relational database query language

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/janmejoykar1807/SQL_project.git
   ```

2. Open in MySQL Workbench: **File → Open Model → select `Final_model.mwb`**

3. Forward-engineer to a live database via **Database → Forward Engineer**

---

## 💡 Concepts Demonstrated

- Entity-Relationship (ER) modeling
- Primary and foreign key constraints
- Table normalization and relational schema design
